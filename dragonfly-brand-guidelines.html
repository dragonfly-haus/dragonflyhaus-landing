Brand guidelines

v1.0 — April 2026

01 — Identity

dragonfly is a software factory. We build special solutions for clients who find us. We don't advertise, we don't pitch. The brand reflects this: geometric, silent, precise.

The mark is a dragonfly constructed from four geometric shards, a body dot, and a tail line terminating in a point. Front wings render at full opacity; back wings at 25% — creating depth from a flat shape.

02 — Logo construction

Anatomy

Front wings: Sharp isoceles triangles, opacity 100%
Back wings: Smaller triangles, offset lower, opacity 25%
Body: Centered circle at wing intersection
Tail: Vertical line with round cap, terminating in a smaller circle
Wordmark: "dragonfly.haus" in monospace, letter-spacing 2px

04 — Typography

One typeface. Monospace only.

Primary: JetBrains Mono
Fallbacks: SF Mono → Fira Code → Cascadia Code → monospace
Weights: 400 (regular), 500 (medium — headings only)
Letter-spacing: 2–4px for brand text, 0 for body copy

Icon-only mark at 16px minimum. Full lockup with wordmark at 120px wide minimum. Below these thresholds the back wings become indistinguishable.

06 — Clear space

Maintain a clear zone around the mark equal to the width of one front wing. No text, graphics, or edges should enter this zone.

07 — Usage rules

Do

Use the mark at provided proportions. Let it inherit currentColor. Place on solid backgrounds only. Use the icon-only variant for small contexts (favicons, avatars, CLI output).

Don't

Don't rotate, stretch, add effects, outlines, shadows, or gradients. Don't place on busy or photographic backgrounds. Don't change the wing opacity ratio. Don't use colors — the mark is monochrome only. Don't add a tagline.

09 — Voice

dragonfly doesn't speak much. When it does:

Tone: Quiet, direct, technical. No exclamation marks.
Casing: All lowercase. "dragonfly" never capitalised.
Domain as name: When the full name is needed, use "dragonfly.haus" — it's the name and the address.
No tagline. No mission statement. No "about" page. The work speaks.
